# Core API and Numerics Tests

This folder covers small, import-light contracts for public VMEC data
structures and numerical helpers:

- namelist/config parsing,
- boundary and coordinate evaluation,
- state/static/profile helper behavior,
- small field/energy identities,
- compatibility-layer behavior.

Full solver behavior belongs in `tests/solvers/`; external-code agreement
belongs in `tests/parity/`.
